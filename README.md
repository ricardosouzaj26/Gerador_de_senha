# Gerador de senha em Python

Este projeto é um **gerador de senhas aleatórias** feito em Python.  
Ele cria senhas fortes combinando números, letras maiúsculas e minúsculas e caracteres especiais.

---

## Funcionalidades

- Gera senhas com:
  - Números  
  - Letras (A–Z, a–z)  
  - Caracteres especiais  
- O usuário escolhe o tamanho da senha  
- Validação de entrada para evitar valores inválidos  

---

## Como usar

1. Tenha o Python 3 instalado.
2. Salve o arquivo como:
   ```bash
   password_generator.py
Execute no terminal:
  ```bash
python password_generator.py
